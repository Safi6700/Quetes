## 🎯 L'essentiel (définitions courtes à apprendre)

1. **Parc informatique** = ensemble des **équipements et logiciels** d'une entreprise (PC, serveurs, imprimantes, licences...)

2. **Gestion de parc** = **entretenir, développer et optimiser** les ressources IT

3. **CMDB** = base de données qui recense tous les **équipements** et leur configuration

4. **GLPI** = logiciel libre pour gérer le **parc + helpdesk** (gestion **passive**)

5. **MDM** (Mobile Device Management) = outil pour gérer les **appareils mobiles** à distance (gestion **active**)

6. **Inventaire** = liste de tous les équipements avec leurs caractéristiques

---

## 🔄 Les 3 axes de la gestion de parc

| Axe | C'est quoi |
|-----|------------|
| **Entretenir** | Maintenir en état de fonctionnement |
| **Développer** | Faire évoluer, renouveler le matériel |
| **Optimiser** | Améliorer l'efficacité, former les users |

---

## 📋 GLPI vs MDM

| | GLPI | MDM |
|--|------|-----|
| **Type** | Gestion **passive** | Gestion **active** |
| **Cible** | Parc fixe (PC, serveurs) | Appareils **mobiles** |
| **Actions** | Inventaire, tickets | Déployer, verrouiller, effacer à distance |

---

## 🔄 Cycle de renouvellement

| Type | Durée |
|------|-------|
| **Matériel acheté** | 4-6 ans |
| **Leasing** | 3 ans |

---

## ⚠️ Piège classique

> **GLPI ≠ MDM**
> - GLPI = inventaire + tickets (passif)
> - MDM = gestion mobiles à distance (actif)
